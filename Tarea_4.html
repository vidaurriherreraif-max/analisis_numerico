Definición 1 Supongamos que \(p^*\) es una aproximación de \(p\). El error real es \(p-p^*\), el error absoluto es \(|p-p^*|\) y el error relativo es

\[\begin{equation} \frac{|p-p^*|}{|p|}, \end{equation}\]

siempre y cuando \(p\neq 0\).

Ejercicio 6 Una variable aleatoria continua \(X\) sigue una distribución ji-cuadrada con \(k\) grados de libertad, lo que se denota como \(X \sim \chi^2(k)\), si su función de densidad de probabilidad es:

\[\begin{equation} f(x) = \frac{1}{2^{k/2} \Gamma(k/2)} x^{k/2-1} e^{-x/2}, \end{equation}\]

para \(x > 0\) y \(k > 0\). Aquí \(\Gamma\) es la función gamma, que generaliza el factorial a los números reales.

El único parámetro de esta distribución es \(k\), que representa los grados de libertad que, aunque puede ser cualquier número real positivo, regularmente son enteros positivos. La función de densidad de probabilidad de la distribución ji-cuadrada es asimétrica y se utiliza comúnmente en pruebas de hipótesis y análisis de varianza.

Los grados de libertad representan el número de variables independientes que se suman para formar la variable aleatoria \(X\). Específicamente:

Si \(Z_1, Z_2, \ldots, Z_k\) son \(k\) variables aleatorias independientes que siguen una distribución normal estándar (es decir, \(Z_i \sim N(0, 1)\) para \(i = 1, 2, \ldots, k\)), entonces la variable aleatoria \(X\) definida como:

\[\begin{equation} X = Z_1^2 + Z_2^2 + \ldots + Z_k^2\sim \chi^2(k) \end{equation}\]

A continuación se muestra la función de densidad de probabilidad de la distribución ji-cuadrada para diferentes valores de \(k\).

Código 
dfs = [3, 5, 7, 10, 12, 15]

x = np.linspace(0, 30, 1000)
plt.figure(figsize=(10, 6))
for df in dfs:
    plt.plot(x, chi2.pdf(x, df), label=f'k={df}')
plt.title('Función de Densidad de Probabilidad de la Distribución Ji-Cuadrada')
plt.xlabel('x')
plt.ylabel('f(x)')
plt.legend()
plt.grid(alpha=0.3)
plt.show()
